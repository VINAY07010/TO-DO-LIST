## How to Use

### Adding Tasks
1. Enter task description in the input field
2. (Optional) Set a due date using the date picker
3. (Optional) Select a category and priority level
4. Click "Add Task"

### Managing Tasks
- **Complete/Undo**: Click the "Complete" button to mark tasks as done
- **Edit**: Click the "Edit" button to modify task details
- **Delete**: Click the "Delete" button to remove tasks

### Searching and Filtering
- Use the search box to find tasks by keyword
- Filter tasks by:
  - Category (All, Work, Personal, Shopping, Health)
  - Priority (All, High, Medium, Low)
  - Status (All, Pending, Completed)

### Dark Mode
- Toggle dark mode using the moon/sun icon in the top right corner
- Preference is saved between sessions

## Technical Details

### Technologies Used
- **HTML5** - Structure and content
- **CSS3** - Styling and animations
- **JavaScript** - Functionality and interactivity
- **localStorage** - Data persistence
- **Browser Notification API** - Task reminders

### No Backend Required
This is a frontend-only application that:
- Stores all data in the browser's localStorage
- Works offline after initial load
- Requires no server or database
- Can be deployed to any static hosting service
